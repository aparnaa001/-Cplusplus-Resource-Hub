C++ Standard Library Reference

C Library

The C Standard Library is a collection of functions and macros that provide essential functionality for C and C++ programs.

  • <cassert>: Diagnostics library for assertions.
  • <cctype>: Functions for character handling.
  • <cerrno>: Error handling macros.
  • <cfenv>: Floating-point environment control.
  • <cfloat>: Constants for floating-point types.
  • <cinttypes>: Formatting macros for integer types.
  • <climits>: Limits for integer types.
  • <cmath>: Mathematical functions.
  • <cstdarg>: Variable argument handling.
  • <cstddef>: Standard definitions and macros.
  • <cstdio>: Input/output functions.
  • <cstdlib>: General utilities (memory allocation, random numbers, etc.).
  • <cstring>: String manipulation functions.
  • <ctime>: Time and date functions.

Containers

Containers are data structures that store collections of objects. The C++ Standard Library provides several container types.

  • <array>: Fixed-size array container.
  • <vector>: Dynamic array container.
  • <deque>: Double-ended queue.
  • <list>: Doubly-linked list.
  • <forward_list>: Singly-linked list.
  • <set>: Collection of unique keys.
  • <map>: Collection of key-value pairs.
  • <unordered_set>: Hash-based set.
  • <unordered_map>: Hash-based map.
  • <stack>: LIFO stack.
  • <queue>: FIFO queue.
  • <priority_queue>: Priority queue.

Algorithms

The C++ Standard Library provides a rich set of algorithms for operations on containers and ranges.

  • Sorting: std::sort, std::stable_sort.
  • Searching: std::binary_search, std::find.
  • Modifying: std::copy, std::transform.
  • Partitioning: std::partition, std::stable_partition.
  • Heap Operations: std::make_heap, std::push_heap.
  • Min/Max: std::min, std::max, std::minmax.
  • Permutations: std::next_permutation, std::prev_permutation.

Utilities

Utility components provide essential functionality for C++ programs.

  • <utility>: Pair and tuple utilities.
  • <memory>: Smart pointers (std::unique_ptr, std::shared_ptr).
  • <functional>: Function objects and binders.
  • <chrono>: Time utilities.
  • <type_traits>: Type traits for metaprogramming.
  • <optional>: Optional object wrapper.
  • <variant>: Type-safe union.
  • <any>: Type-safe container for single values.
