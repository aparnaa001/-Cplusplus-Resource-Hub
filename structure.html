C++ Program Structure

Basic Syntax

A C++ program consists of the following components:

  • Preprocessor Directives: Include headers like #include <iostream>.
  • Main Function: The entry point of the program, defined as int main().
  • Statements: Executable instructions like std::cout << "Hello, World!";.
  • Comments: Use // for single-line or /* ... */ for multi-line comments.
  • Namespaces: Use std to avoid typing std:: repeatedly.
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!";
    return 0;
}

Functions

Functions are blocks of code that perform specific tasks. Key concepts include:

  • Function Declaration: Define the function signature before use.
  • Function Definition: Implement the function logic.
  • Parameters and Arguments: Pass data to functions.
  • Return Values: Functions can return values using return.
#include <iostream>
using namespace std;

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 3);
    cout << "Sum: " << result;
    return 0;
}

Classes and Objects

Classes and objects are the foundation of Object-Oriented Programming (OOP) in C++.

  • Class Definition: Use class to define a blueprint for objects.
  • Objects: Instances of a class.
  • Encapsulation: Use private and public to control access.
  • Constructors and Destructors: Initialize and clean up objects.
#include <iostream>
using namespace std;

class Rectangle {
private:
    int width, height;
public:
    Rectangle(int w, int h) : width(w), height(h) {}
    int area() {
        return width * height;
    }
};

int main() {
    Rectangle rect(5, 10);
    cout << "Area: " << rect.area();
    return 0;
}
